Installation Guide

Linux mcpelauncher-manifest

1. Extract material.bin files from shader mcpack / build materials manually

2. Move these files to data root mcpelauncher/versions/1.20.x/assets/renderer/materials/. (Make sure to backup all files in this folder first)

3. Import the resource pack and activate it in global resources.

Windows

1. Download BetterRenderDragon to enable MaterialBinLoader.

2. Extract zip file and run LaunchMinecraft.bat.

3. Import the resource pack and activate it in global resources.

Android

1. Download and install Minecraft Patched Apk

2. Import the resource pack and activate it in global resources.

iOS

1. Jailbreak yours iOS (not recommended).

2. Sideload modified Minecraft.

3. Import the resource pack and activate it in global resources. (use android pack)
